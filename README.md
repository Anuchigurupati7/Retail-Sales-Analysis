# Retail Sales Data Analysis

## Overview
This project analyzes retail sales data using Python to identify trends, patterns, and basic business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel
- Tableau

## Project Files
- `data/sample_data.csv` — sample retail sales dataset
- `data/analysis.py` — Python script for basic analysis and visualization

## Key Tasks
- Loaded and explored retail sales data
- Grouped sales by category
- Generated a simple bar chart for sales comparison
- Practiced data analysis and visualization in Python

## Outcome
This project demonstrates basic data analysis skills and how Python can be used to generate simple business insights from sales data.
